# Final Project: Emotion Detection Application

This repository contains the Final Project for the IBM Developer Skills Network course.

## Project Description
This project implements an Emotion Detection application using the Watson NLP library. The application analyzes text input and returns emotions such as joy, anger, sadness, fear, and disgust.

## Features
- Emotion detection using Watson NLP
- Flask-based web deployment
- Error handling for invalid inputs
- Unit testing and validation
- Static code analysis using pylint
